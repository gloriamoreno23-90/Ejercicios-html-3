Conceptos HTML

Párrafos:

El elemento <p> es el apropiado para distribuir el texto en párrafos en un documento html. Sus etiquetas son: <p> para la apertura y </p> para el cierre. Todo el texto contenido entre las etiquetas se visualizará en la página web

Anchor:

El elemento <a> permite crear un enlacce a otras páginas de internet, archivos, direcciones de correo o cualquier otra URL.

Title:

Este es un atributo global que contiene un texto que describe el elemento que representa. Este texto se visualiza como una pequeña ventana emergente.
